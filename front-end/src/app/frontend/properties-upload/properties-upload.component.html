Exporting data into a CSV file before importing into cmaIQ allows for the ability to manually enter and manipulate data which has the potential for errors. cmaIQ does not make any representations or guarantees regarding the data, the use or the results derived, including without limitation, in terms of correctness, accuracy, or reliability.

For the cmaIQ Residential Pricing, Investment & Analytics Tool to generate an accurate analysis of home value, only the Active, Pending and Closed property listings will be used. If you included other listing types (ie. Expired, Canceled, Withdrawn, etc.) this data will not be considered in the cmaIQ analysis.
